## 🚀 Setup & Execution

### Prerequisites
- Java 11 or higher
- Maven 3.6+

### Installation
```bash
# Clone this repository
git clone https://github.com/Uttam1225/DailyAutomation-2026-05-28.git
cd DailyAutomation-2026-05-28

# Install Playwright browsers
mvn exec:java -e -D exec.mainClass=com.microsoft.playwright.CLI -D exec.args="install"
```

### Run Tests
```bash
# Run all tests
mvn clean test

# Run in headless mode
mvn clean test -Dheadless=true

# Run with specific base URL
```
